# Arbitrage Strategy

אסטרטגיית ארביטראז' בין שווקים היררכיים.

## אסטרטגיה

מזהה הבדלי מחירים בין שווקים קשורים ומבצע עסקאות דו-כיווניות.

**⚠️ שימוש במחירים אמיתיים:**
האסטרטגיה משתמשת ב-orderbook האמיתי (ASK לקנייה, BID למכירה) ולא במחירי Mid,
כדי לחשב רווח אמיתי אחרי Spread.

## דוגמה

```
שוק 1: Bitcoin above $100k by Dec 31?
  ASK (לקנייה): $0.45

שוק 2: Bitcoin above $100k by Dec 15?
  BID (למכירה): $0.50

ארביטראז' אמיתי:
קנה בשוק 1 @ $0.45 (ASK)
מכור בשוק 2 @ $0.50 (BID)
רווח: $0.05 (11.1%)
```

## הרצה

```bash
python -m strategies.arbitrage.strategy
```

## הגדרות

- `min_profit_pct`: אחוז רווח מינימלי (2%)
- `max_hours_until_close`: מקסימום שעות עד סגירה (24)
