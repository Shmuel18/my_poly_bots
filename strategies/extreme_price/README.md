# Extreme Price Strategy

אסטרטגיה לקנייה במחירים קיצוניים והמתנה להכפלה.

## אסטרטגיה

1. **סריקה** - מחפש שווקים במחירים נמוכים מאוד (0.4 cents)
2. **כניסה** - קונה במחיר נמוך
3. **יעד** - ממתין להכפלת המחיר (0.8 cents)
4. **יציאה** - מוכר ברווח 100%

## דוגמה

```
קנייה: 250 יחידות @ $0.004 = $1.00
מכירה: 250 יחידות @ $0.008 = $2.00
רווח: $1.00 (100%)
```

## הרצה

```bash
python -m strategies.extreme_price.strategy
```

## הגדרות

- `buy_threshold`: מחיר מקסימלי לקנייה ($0.004)
- `sell_multiplier`: מכפיל למכירה (2.0)
- `min_hours_until_close`: מינימום שעות עד סגירה (1)
- `portfolio_percent`: אחוז מהתיק (0.5%)
